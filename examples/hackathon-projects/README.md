### Pipe It!! - Invoice Factoring Platform
[DoraHacks Submission 25577](https://dorahacks.io/buidl/25577)

Pipe It!! is a Base-powered invoice factoring platform with intelligent yield generation. It enables businesses to borrow against their invoices, while allowing investors to earn APY through AI-driven trading strategies.

**How It Works:**
1. **Invoice Submission:** Businesses register and submit invoices via the frontend
2. **Factoring Offer:** Pipe It!! evaluates risk & sentiment and offers an advance
3. **Advance Payout:** If accepted, ETH is transferred from the Flooz Vault
4. **Idle Funds Trading:** Unused funds are allocated using AI-generated strategies
5. **Profit Reporting:** Profits are distributed proportionally to vault participants

**Token Metrics Integration:**
- Uses Token Metrics for market-responsive pricing using real-time sentiment data
- AI-generated trading strategies for idle funds

### SwAutomate Telegram Trading Agent Bot
[DoraHacks Submission 25562](https://dorahacks.io/buidl/25562)

A Telegram Agent Bot that enables secure automated trading with encrypted receipts for tax and accounting purposes.

**Problem Solved:**
Retail and casual crypto traders often lack the time, technical knowledge, or discipline to follow through on effective trading strategies. Most bots require custom coding or lack flexibility. Additionally, tracking trades for tax purposes is tedious and error-prone.

**Solution:**
The bot leverages ViaLabs to fetch real-time trading signals from Token Metrics using AgentKit, combined with ElizaOS for bot orchestration. It stores encrypted trade receipts on-chain via Request Network, ensuring verifiable proof of trade history for tax and compliance.

**Key Integrations:**
- Base, for using the AgentKit
- Token Metrics, for using their signals within the trading algorithm
- ViaLabs, to secure the oracle that the trading bot analyzes
- Request Network, to store and fetch encrypted receipt trading information for tax and accounting purposes

**Links:**
- [GitHub Repository](https://github.com/armsves/EthBucharest2025TradingAgentBot.git)
- [Demo Video](https://youtu.be/gCq0HnZ90xs)
