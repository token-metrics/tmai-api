# Token Metrics API Hackathon Projects

This directory contains examples and resources from various hackathons and community projects that utilize the Token Metrics AI API. These projects demonstrate real-world applications and integrations of the API for cryptocurrency analysis, trading, and portfolio management.

## Top Projects Showcasing Token Metrics API

The following projects are sorted in descending order of quality based on their Token Metrics API integration, feature completeness, innovation, and overall implementation.

### 1. Let Me Know - Multi-Agent Crypto Portfolio Management

Let Me Know is an intelligent crypto portfolio management platform designed to empower investors with actionable, AI-driven insights. By seamlessly connecting to users' wallets, the platform analyzes token holdings in real-time and leverages a suite of specialized AI agents to deliver clear recommendations.

**AI Trading Agents:**
- **Bounce Hunter:** Identifies key support and resistance levels to anticipate price movements
- **Crypto Oracle:** Analyzes market sentiment and token momentum for informed predictions
- **Momentum Quant Agent:** Delivers quantitative momentum analysis to capture trends
- **SMA Agent:** Uses moving average crossover strategies to detect bullish and bearish signals
- **Portfolio Manager:** Synthesizes insights from all agents to provide balanced recommendations

**Token Metrics Integration:**
- Accesses live market data for precise and timely analysis
- Powers AI-driven trading signals and market insights
- Enables real-time portfolio analysis and recommendations

### 9. Pipe It!! - Invoice Factoring Platform

Pipe It!! is a Base-powered invoice factoring platform with intelligent yield generation. It enables businesses to borrow against their invoices, while allowing investors to earn APY through AI-driven trading strategies powered by Token Metrics.

**How It Works:**
1. **Invoice Submission:** Businesses register and submit invoices via the frontend
2. **Factoring Offer:** Pipe It!! evaluates risk & sentiment and offers an advance
3. **Advance Payout:** If accepted, ETH is transferred from the Flooz Vault
4. **Idle Funds Trading:** Unused funds are allocated using AI-generated strategies
5. **Profit Reporting:** Profits are distributed proportionally to vault participants

**Token Metrics Integration:**
- Uses Token Metrics for market-responsive pricing using real-time sentiment data
- AI-generated trading strategies for idle funds
